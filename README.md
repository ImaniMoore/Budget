# Imani's Budget Tracker

A simple web application to track your income, expenses, and total budget. Built using **HTML**, **SASS**, and **JavaScript** with object-oriented programming concepts.

## Features

- Add **income** and **expense** transactions with description and amount.
- Automatically calculate **total income**, **total expenses**, and **remaining budget**.
- Input validation prevents empty or invalid entries.
- Responsive layout for **desktop, tablet, and mobile**.
- Uses **object-oriented programming** (OOP) principles:
  - `Transaction`, `Income`, `Expense`, and `Budget` classes.
- Visually appealing UI styled with **SASS**
- Hover animations for transaction cards.
